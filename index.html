Thoughts on Progressive Enhancement

From what I interpreted from the article on progressive enhancement, I think progressive enhancement is a way to describe the different layers of a web design. In the article in described progressive enhancement as an peanut M&M, which I think is a perfect way to describe web desgin and how it layered. The peanut in the M&M is is the HTML, which is the foundation of the M&M and in the case of the M&M the chocolate and the shell are layered over the peanut and without the peanut it, would not be a peanut M&M.

Article goes on to describe the chocolate as the CSS of web design, which is true. CSS brings out the HTML, making the web appeal and bring awe to the veiwer, and just as the chocolate in the peanut M&M, the chocolate combined with the peanut allows for a memorable and fiending enjoyment with the site. Lastly, the shell of the peanut M&M is Java Script and this gives the viewer of the website a more interactive website experience. The shell gives the peanut M&M a way to intereact and keep the chocolate and peanut together.
